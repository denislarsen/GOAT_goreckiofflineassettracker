How to read this

Green — Stable. No early-warning signs triggered.
Yellow — Watch. One or two indicators are flashing early weakness. Worth monitoring, not yet acting on.
Red — Alert. A confirmed drop, downtrend, or multiple simultaneous warning signs. Consider reviewing this position.

Each asset class (except Cash) is scored on four indicators: drawdown from its recent 20-day high, the 10/30-day moving-average trend, 5-day momentum, and a volatility spike on down days. Cash carries no market price risk, so it is always shown as stable.

Settings

By default this dashboard runs on bundled sample data so it works fully offline. To see live prices, add a free Alpha Vantage API key (get one at alphavantage.co) — it is stored only in this browser.

Free keys are rate-limited (roughly 25 requests/day). The dashboard caches results and spaces out requests automatically.
